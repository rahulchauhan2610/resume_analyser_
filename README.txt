# 📄 Resume Analyzer

## 🎯 **Project Overview**
The **Resume Analyzer** is a web application built with **Streamlit**, powered by **SpaCy** and **Pyresparser** to extract and analyze information from resumes. It automates the resume screening process by extracting key details such as name, contact information, education, skills, and experience.

---

## 🚀 **Features**
- ✅ Upload resumes in **PDF/Docx** format.
- 📊 Extracts and displays:
  - Name and contact details
  - Education and experience
  - Skills and certifications
- 📚 Generates a structured summary of the resume.
- 📉 Provides insights to evaluate candidates efficiently.

---

## 🛠️ **Tech Stack**
- **Frontend:** Streamlit  
- **Backend:** Python  
- **Libraries Used:**
  - `SpaCy` – for NLP and text extraction.
  - `Pyresparser` – to parse resumes and extract key information.
  - `Pandas` – for data manipulation.
